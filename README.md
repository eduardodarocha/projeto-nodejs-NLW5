## üöÄ Getting started

Clone the project and access the folder

```bash
$ git clone https://https://github.com/eduardodarocha/projeto-nodejs-NLW5.git

$ cd projeto-nodejs-NLW5
```
Follow the steps below

```bash
# Install the dependencies
$ yarn

# Start the project
$ yarn dev
```
